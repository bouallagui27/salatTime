# 🕌 Aw9at Salat - Prayer Times Project

This is a small project that displays daily prayer times for all Tunisian cities along with the Hijri date.  
Purpose: Learn how to work with an **API**, update the user interface dynamically, and use **JavaScript** with **TailwindCSS**.

---

## 📌 Features

- Select a Tunisian city from a dropdown menu.
- Display prayer times: Fajr, Dhuhr, Asr, Maghrib, Isha.
- Show the Hijri date dynamically.
- Times update instantly when the city is changed.
- Responsive and clean design using TailwindCSS.

---

## 🛠️ Tools Used

- HTML / CSS / JavaScript
- [TailwindCSS](https://tailwindcss.com/) for styling
- [Aladhan API](https://aladhan.com/prayer-times-api) to fetch prayer times and Hijri date

---

## 🚀 How to Use

1. Open `index.html` in your browser.
2. Select a Tunisian city from the dropdown.
3. Prayer times and the Hijri date will appear instantly.
